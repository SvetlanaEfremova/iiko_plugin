Для сборки плагина потребуется .Net Framework версии 4.7.2

Сборка плагина через Visual Studio:
Откройте Visual Studio.
В меню "Файл" выберите "Открыть" -> "Проект/Решение" или нажмите Ctrl+Shift+O.
Выберите файл решения iiko_plugin_test_task.sln
Выберите в меню "Сборка" опцию "Пересобрать решение".

Сборка плагина с помощью командной строки:
Запустите командную строку.
Выполните команды: 
cd путь/до/папки/решения, где решение - файл iiko_plugin_test_task.sln
dotnet restore
dotnet build

Сборка решения создаст файл iiko_plugin_test_task.dll в папке bin/Debug, который является собранным плагином.

Для использования плагина в iikoFront скопировать .dll-файл в поддиректорию директории Plugins, находящейся в каталоге установки iikoFront.
